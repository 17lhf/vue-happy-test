# vue-happy-test
本项目为作者实现前端具体功能期间，附带记录的笔记。主要是留作以后阅览。也顺带补充了一些网上很少见到的实现方式。请先耐心阅读本文，再行阅读代码。<br />
本项目启动方式：npm run dev <br/>
后端代码：[点击前往](https://github.com/17lhf/happyTest)
